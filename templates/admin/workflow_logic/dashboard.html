How Workflow Logic Works
  1. Field Conditions: Create if/else logic between form fields
  2. Actions: Show, hide, require fields based on other field values
  3. Operators: Use equals, contains, greater than, etc.
  4. Order: Conditions execute in order - lower numbers first
  5. Testing: Always test your logic before deploying
Example Workflow:

IF "Order Type" equals "Custom"
THEN Show "Custom Specifications" field
AND Make it required
